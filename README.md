# Personal Profile Page

This is a **simple personal profile webpage** created using **HTML and CSS**. It demonstrates the use of various web development concepts such as inline CSS styling, Flexbox, CSS Grid, forms, and responsive design. This project is suitable for beginners to practice building a personal webpage layout without using any frameworks.

---

## Features

- **Header Section**
  - Displays the full name with custom font, colors, and background image.
  - Includes decorative borders and text styling.

- **Profile Image**
  - Centered profile image using Flexbox.
  - Styled with a fixed width and proper spacing.

- **Bio Section**
  - Short paragraph about the user.
  - Styled with background color, borders, padding, and centered text.

- **Interests / Skills Section**
  - Uses **CSS Grid** to layout 4 interest cards.
  - Each card has background color, padding, rounded corners, and centered text.
  - Responsive design ensures it looks good on mobile devices.

- **Contact Section**
  - Includes a **contact form** with Name, Email, and Message fields.
  - Email link uses `mailto:` for easy contact.
  - Styled using padding, font styles, alignment, and button design.

- **Footer**
  - Displays name and copyright text.
  - Centered content with background color and padding.

---

## Technologies Used

- **HTML5**
- **CSS3** (inline styling)
- **Google Fonts (Poppins)**

---

## Usage

1. Clone or download this repository.
2. Open `index.html` in a web browser.
3. Replace the profile image (`coder.jpg`) with your own images if desired.
4. Update contact details and interests as needed.
